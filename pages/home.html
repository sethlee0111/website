• Ongoing Projects

Decentralized Learning in Pervasive Computing

My primary focus is on developing algorithms, frameworks, and tools that help us extract machine intelligence from data collected by mobile and IoT devices.
Mobile and IoT devices are incredibly pervasive in our daily lives, generating vast amounts of data. This explosion of data has led to exciting advancements in machine learning and new applications that were previously impossible using traditional approaches.
The question I tried to answer throughout my PhD is that whether these devices can collaborate each other to learn from the data that is decentralized, as opposed to training a model with all the data on a single machine. During my PhD, I have proposed a new decentralized learning approach called Opportunistic Collaborative Learning which aims to achieve model personalization in a completely decentralized way. I have also developed a dynamic and adaptive model compression technique for decentralized learning, and a simulation framework for fast and scalable testing for decentralized learning algorithms.
